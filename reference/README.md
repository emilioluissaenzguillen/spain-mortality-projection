# Reference Material

This folder contains non-input documentation.

## Most important reference files

- `INE/mortality_projection_note.md`
  Working note summarizing the methodology evidence gathered from INE material.
- `INE/emails`
  INE replies and clarifications.
- `INE/methodology`
  Methodology PDFs and related background documents.
- `UN`
  UN and longer-horizon reference material reserved for the later `2100` extension.

These files are used to understand and document the methodology.
They are not treated as direct machine inputs in the active replication script.
